## CHECKLISTS

**1. Register Page:**
- Открывается ли страница регистрации?
- Поля заполнение
- Валидные форматы для: Email, First and Last name
- Валидность пароля: Обязательность, Минимум длина, Подверждение пароля.
- Регистрация с валидными данными успешна
- Регистрация с уже существующим email невозможна
- Сообщение об ощибках

**2. Log in/Log out pages:**
- Открывается ли страницы Log in?
- Логин с валидными данными успешен
- Возможно ли: логин с невалидным email, логин с не верным паролем?
- Пустые поля не должны приниматься
- Log out должен быть успешен

**3. Search:**
- Есть ли поле Поиска?
- Результаты поиска релеванты к запросу
- Поиск по несуществующему товару возвращает результат “No products found”

**4. Product Page:**
- Страница товара открывается
- Должны отоброжаться: Название, цена, описание, рейтинг товара, кнопка "Add to Cart"
- Количество можно изменить
- Кнопка "Add To Cart" работает успешно
- Некоректное количество не возможна
- Товар добавлен в корзину

**5. Checkout:**
- Чек доступен только авторизоанному пользователю
- Обьязательные поля отмечены: Город, адрес, дом, номер телефона,
- Принимание правил сервиса
- Подверждение заказа
- Успешная создание заказа

**6. UI/UX Interface:**
- На все кнопки можно нажать и они работают
- Ошибки отоброжаються корректно
- Ссылки не ведут на не найденную страницу (404 Not Found)
